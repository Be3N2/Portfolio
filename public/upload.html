Driving Data Upload

Upload a file to get processed in the system.


After uploading a file and submitting the response will include a json object with file name and size in bytes.
Ex: { "name": "largePhoto.JPG", "size": 1663142}
Project made with multer library for file management.


Project made by Ben Wakefield for Freecodecamp. Source code and more info available on Github.
Documentation on the multer library.
